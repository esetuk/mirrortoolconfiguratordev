MirrorToolConfigurator

This tool is designed to help you configure command line parameters and arguments for the ESET MirrorTool. For help setting up the MirrorTool please see the help links at the bottom of the page.

Start by settings the filters to your desired platform and parameters, then configure each of your desired parameter arguments. Finally, either copy the commands to clipboard or download a script which you can run in the same directory as the MirrorTool.
